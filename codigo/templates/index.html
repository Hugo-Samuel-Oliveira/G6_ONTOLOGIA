Assistente Virtual Semântico BIM

Faça uma pergunta ou clique duas vezes em um nó para explorá-lo.

Como Usar

Para consultar um objeto, use aspas simples no nome, como 'floor'.

Exemplos de Perguntas:

  • Qual o material do 'floor'?
  • Onde está a parede 'house - outer wall - house left'?
  • Qual o tipo do 'floor'?
  • Mostre o grafo inteiro

Interação com o Grafo:

  • **Clique duplo** em um nó para focar nele e ver suas conexões.
  • Clique no botão **Restaurar Grafo** para voltar à visualização original.
